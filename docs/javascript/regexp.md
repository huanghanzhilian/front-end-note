### 电子邮箱
``` javascript
/^[0-9a-z][0-9a-z\-\_\.]+@([0-9a-z][0-9a-z\-]*\.)+[a-z]{2,}$/i
```

### 固定电话
``` javascript
/^0[0-9]{2,3}[2-9][0-9]{6,7}$/
```

### 手机号码
``` javascript
/^(\+?0{0,2}86([\ |\-])?)?1[3|4|5|7|8][0-9]{9}$/
```

### 身份证号码
``` javascript
/^(\d{15}$|^\d{18}$|^\d{17}(\d|X|x))$/
```

### ip地址
``` javascript
/^((25[0-5])|(2[0-4]\d)|(1\d\d)|([1-9]\d)|\d)(\.((25[0-5])|(2[0-4]\d)|(1\d\d)|([1-9]\d)|\d)){3}$/
```

### 域名
``` javascript
/^([a-zA-Z0-9][-a-zA-Z0-9]{0,62}\.)+([a-zA-Z]{0,63})\.?$/
```
